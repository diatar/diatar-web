Hangfájlok kezelése

<< Vissza   Tovább >>

 

Az énekrend listáiban egy kék kottafej jelzi, hogy az énekhez hang tartozik. A jel szürke, ha nincs hang megadva. Az énekrend módosításakor rákattinthatunk a jelre, így letilthatjuk az adott hangot.

Mellette kis zöld nyíl mutatja, hogy lejátszás után a program váltani fog a következő diára. Ha ott is van megadva zene, akkor az is lejátszásra kerül, majd (ha van zöld nyíl), továbblép a program az utána levő diára és így tovább. Az énekrend módosításakor a nyílra rákattinthatunk, így a továbbítást ki/be kapcsolhatjuk bármelyik dia esetén.

Az énekrend módosításakor az összeállított lista alján levő [Tulajdonságok] gombra kattintva megadhatjuk a hangfájlt, amelyet le szeretnénk játszani.
